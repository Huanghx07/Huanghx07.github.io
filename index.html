Vue中export default与export的区别

  • 在ES6中导入模块，使用 import 模块名称 from ‘模块标识符’
  • 在 ES6 中，使用 export default 和 export 向外暴露成员：
  • export default只允许向外暴露一次
  • export可以向外暴露多个成员
  • 使用 export 导出的成员，必须严格按照 导出时候的名称，来使用 { } 按需接收，不需要，可以不在{}中定义
  • 在一个模块中，可以同时使用export default和export暴露成员
